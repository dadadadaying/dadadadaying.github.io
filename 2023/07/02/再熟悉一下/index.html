再熟悉一下

又来了

hexo的基本命令：

hexo new ‘postName’ #新建文章

hexo new page ‘pageName’ #新建页面

hexo generate #生成静态页面至public目录

hexo server #开启预览访问端口（默认端口4000，’ ctrl + c ‘ 关闭server）

hexo deploy #部署到GitHub

hexo help #查看帮助

hexo version #查看Hexo的版本

命令可以缩写：

hexo n == hexo new

hexo g == hexo generate

hexo d == hexo deploy

搭建好博客后，日常中创建和部署博客的流程：

0.创建一个保存博客的存放目录，如D:\blog

​ 在桌面右键git bash here进入终端

​ 进入D:\blog

​ 键入命令 hexo init ，初始化博客。然后可以看到博客存放目录出现了许多文件夹，大概如下图所示：

1.新建博客 hexo new ‘博客名’

​ 在目录source/_posts下就会出现 ‘博客名.md’文件，通过这个md文件写博客

2.写完以后，部署到github上

​ hexo g 生成html文件

​ hexo d 上传到github

这样就可以了
